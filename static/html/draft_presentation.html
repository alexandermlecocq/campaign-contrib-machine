Mapping Campaign Donations

to Predict Future Political Donors

Alexander Lecocg, Brickey LeQuire, Stephen Monteiro

Goal:

Drawing on publicly available FEC Campaign Contribution datasets from the 2016 Election cycle and ACS 5-Year Census estimates at the zip code level, this project uses machine learning techniques to identify how campaign finance influences elections, and to predict future campaign donors.
